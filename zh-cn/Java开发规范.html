项目结构

这里采用maven构建，MVC结构，项目结构如下：

├── src
│   ├── main
│   │   ├── java
│   │   │   └── com
│   │   │       └── example
│   │   │           └── demo
│   │   │               ├── controller
│   │   │               ├── entity
│   │   │               ├── mapper
│   │   │               ├── service
│   │   │               └── DemoApplication.java
│   │   └── resources
│   │       ├── application.properties
│   │       ├── static
│   │       └── templates
│   └── test
└── pom.xml

其中，controller用于存放控制层代码，entity用于存放实体类，mapper用于存放mapper接口，service用于存放业务逻辑代码，DemoApplication.java是项目启动类，application.properties是项目配置文件。
